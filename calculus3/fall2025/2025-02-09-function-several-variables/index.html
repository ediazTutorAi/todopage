Definition
Function of Two Variables
A function of two variables $z=f(x,y)$ maps each ordered pair $(x,y)$ in a subset $D$ of the real plane $\mathbb{R}^2$ to a unique real number $z$. The set D is called the domain of the function. The range of f is the set of all real number z that has at least one ordered pair $(x,y)$ in D such that $f(x,y)=z$.

Level Curves
Definition
Given a function $f(x,y)$ and a number c in the range of f, a level curve of a function of two variables for the value c is defined to be the set of points satisfying the equation $f(x,y)=c$.
Another useful tool for understanding the graph of a function of two variables is called a vertical trace, which are the curves obtained by intersecting the graph of f with the xz- or yz-plane, or any plane parellel to them.
Definition of vertical trace: Consider a function $z=f(x,y)$ with domain $D \subset \mathbb{R}^2$. A vertical trace of the function can be either the set of points that solves the equation $f(a,y)=z$ for a given constant $x=a$ or $f(x,b)=z$ for a given constant $y=b$.
Functions of More than two variables
A function of threee variables is a function of the form $w=f(x,y,z)$. The same definitions of domain and range apply in this case.
Definition: Given a function $f(x,y,z)$ and a number c in the range of f, a level surface of a function of three variables is defined to be the set of points satisfying the equation $f(x,y,z)=c$.
