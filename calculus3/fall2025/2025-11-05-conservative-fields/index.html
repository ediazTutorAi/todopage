Curves and Regions
Definition
Curve C is a closed curve if there is a parameterization $\vec{r}(t)$,$a\leq t \leq b$ of C such that the parameterization traverses the curve exactly once and $\vec{r}(t)$, $a\leq t \leq b$ of C such that $\vec{r}$ is one-to-one over $(a,b)$. It is possible for $\vec{r}(a)=\vec{r}(b)$, meaning that the simple curve is also closed.
Definition
Connected regions
A region D is a connected region if, ofr any two points $P_1$ and $P_2$, there is a path from $P_1$ to $P_2$ with a trace contained entirely inside D. A region D is a simply connected region if D is connected for any simple closed curve C that lies inside D, and curve C can be shrunk continuously to a point while staying entirely inside D. In two dimensions, a region is simply connected if it is connected and has no holes.
Fundamental Theorem for Line Integrals
Let C be a piecewise smooth curve with parameterization $\vec{r}(t)$, $a\leq t \leq b$. Let f be a function of two or three variables with first-order partial derivatives that exist and are continuous on C. Then,
$$\int_{C} \nabla f \cdot d\mathbf{r}=f(\vec{r}(b))-f(\vec{r}(a))$$
Path Independent
Let $\vec{F}$ be a vector field with domain D. The vector field $\vec{F}$ is independent of path or path independent if
$$\int_{C_1} \mathbf{F} \cdot d\mathbf{r}=\int_{C_2} \mathbf{F} \cdot d\mathbf{r}$$ for any paths $C_1$ and $C_2$ in D with the same initial and terminal points.
Theorem 6.8
Path Independence of Conservative Fields
If $\vec{F}$ is a conservative vector field, then $\vec{F}$ is independent of path.
Theorem 6.9
The Path Independence Test for Conservative Fields
If $\vec{F}$ is a continuous vector field that is independent of path and the domain D of $\vec{F}$ is open and connected then $\vec{F}$ is conservative.

Finding a Potential Function for a Conservative Vector Field $\vec{F}(x,y)=$
1. Integrate P with respect to x. This results in a function of the form $g(x,y)+h(y)$, where $h(y)$ is unknown.
2. Take the partial derivative of $g(x,y)+h(y)$ with respect to y, which results in the function $g_y(x,y)+h'(y)$
3. Use the equation $g_y(x,y)+h'(y)=Q(x,y)$ to find $h'(y)$
4. Integrate $h'(y)$ to find $h(y)$
5. Any function of the form $f(x,y)=g(x,y)+h(y)+C$, where C is a constant, is a potential function for $\vec{F}$.
