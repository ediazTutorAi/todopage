Arc-Length Formulas
The arc length of a curve $C$ defined by $\mathbf{r}(t)= f(t)\mathbf{i}+g(t)\mathbf{j}+h(t)\mathbf{k}$ for $a \leq t \leq b$ is given by $$L=\int_a^b \sqrt{f'(t)^2+g'(t)^2+h'(t)^2}dt=\int_a^b \|\mathbf{r}'(t)\|dt$$
Arc Length Parameterization
Definition
A smooth curve $C$ with position vector $\mathbf{r}(t)$, $a \leq t \leq b$, is parameterized by arc length if $$\|\mathbf{r}'(t)\| = 1$$ Consider the example of a circle of radius 1 centered at the origin with two different parametrerizations.
Curvature
Definition
Let C be a smooth curve in the plane of in space given by $\mathbf{r}(s)$ where $s$ is the arc length parameter. The curvature of C is given by $$\kappa(s) = \|dT/ds\|$$ where $T(s)$ is the unit tangent vector defined as $$T(s) = \frac{dr}{ds}$$

The curvature of a curve is a measure of how much the curve bends. Give examples.
Alternative Formula for Curvature
If C is a smooth curve given by $\mathbf{r}(t)$, then the curvature of C is given by $$\kappa(t) = \frac{\|T'(t)\|}{\|r'(t)\|}$$ If C is a three-dimensional curve, then the curvature of C is given by $$\kappa(t) = \frac{\|r'(t) \times r''(t)\|}{\|r'(t)\|^3}$$ If C is a plane curve given by $y=f(x)$, then the curvature of C is given by $$\kappa(x) = \frac{|f''(x)|}{(1+[f'(x)]^2)^{3/2}}$$ Give the idea of the proof of the first formula.
The normal and binormal vectors
Let C be a three-dimensional smooth curve represented by $\mathbf{r}(t)$ over an open interval I. If $\vec{T}'(t) \neq 0$, then the principal unit normal vector at t is defined to be $$N(t) = \frac{T'(t)}{\|T'(t)\|}$$ The binormal vector at t is defined as $$B(t) = T(t) \times N(t)$$ where $T(t)$ is the unit tangent vector.
Motion in Space
For motion in space we just need to remember that the velocity vector is derivative of the position vector and the acceleration vector is the derivative of the velocity vector. And we use Calculus over vector-valued functions as we had consider in previous sections.
