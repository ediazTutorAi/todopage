Critical Points
Definition
Let \(z=f(x,y)\) be a function of two variables that is defined on an open set containing the point \((x_0,y_0)\). The point \((x_0,y_0)\) is called a critical point of \(f\) if one of the two following conditions holds:
1. \(f_x(x_0,y_0)=0\) and \(f_y(x_0,y_0)=0\)
2. At least one of the partial derivatives \(f_x\) and \(f_y\) does not exist at \((x_0,y_0)\).
Local maximum
Definition
Let \(z=f(x,y)\) be a function of two variables that is defined and continuous on an open set containing the point \((x_0,y_0)\). Then f has a local maximum at \((x_0,y_0)\) if: \[f(x,y) \leq f(x_0,y_0)\] for all points \((x,y)\) in some open disk centered at \((x_0,y_0)\). The number \(f(x_0,y_0)\) is called the local maximum value of \(f\). If the preceding inequality holds for all points \((x,y)\) in the domain of \(f\), then \(f\) has an absolute maximum at \((x_0,y_0)\) and \(f(x_0,y_0)\) is called the absolute maximum value of \(f\).
Local Minimum
Definition
Similarly, \(f\) has a local minimum at \((x_0,y_0)\) if: \[f(x,y) \geq f(x_0,y_0)\] for all points \((x,y)\) in some open disk centered at \((x_0,y_0)\). The number \(f(x_0,y_0)\) is called the local minimum value of \(f\). If the preceding inequality holds for all points \((x,y)\) in the domain of \(f\), then \(f\) has an absolute minimum at \((x_0,y_0)\) and \(f(x_0,y_0)\) is called the absolute minimum value of \(f\).
We refer to local maxima and minima collectively as local extrema.
Theorem 4.16
Fermat's Theorem for Functions of Two Variables
Let \(z=f(x,y)\) be a function of two variables that is defined and continuous on an open set containing the point \((x_0,y_0)\). Suppose that \(f_x\) and \(f_y\) exist at \((x_0,y_0)\). If \(f\) has a local maximum or local minimum at \((x_0,y_0)\), then \(f_x(x_0,y_0)=0\) and \(f_y(x_0,y_0)=0\); that is, the point \((x_0,y_0)\) is a critical point of \(f\).

Equation of the Tangent Plane
Using the Gradient
\[ z - z_0 = f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0). \] Here \( f_x \) and \( f_y \) are partials at \( (x_0,y_0) \).
Linear Approximation
Tangent Plane as \(L(x,y)\)
\[ L(x,y) = z_0 + f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0). \]
