2.3 Matrix Products

MATH 3330 · Dr. Esteban Diaz
Definition 2.3.1
Matrix multiplication
a. Let B be an $n \times p$ matrix and A be an $q \times m$ matrix. The product $BA$ is defined if and only if $p = q$.
b. If B is $n \times p$ and A is $p \times m$, then the product $BA$ is defined as the matrix of the linear transformation $T(\vec{x}) = B(A\vec{x})$. This means that $T(\vec{x}) = B(A\vec{x})=(B A)\vec{x}$ for all $\vec{x}$ in $\mathbb{R}^m$. The product $BA$ is an $n \times m$ matrix.
Theorem 2.3.2
The columns of the matrix product
Let B be an $n \times p$ matrix and A be an $p \times m$ matrix with columns $\vec{v}_1, \vec{v}_2, \ldots, \vec{v}_m$. Then, the product $BA$ is given by: $$BA = \begin{pmatrix} B\vec{v}_1 & B\vec{v}_2 & \cdots & B\vec{v}_m \end{pmatrix}=\begin{pmatrix} B\vec{v}_1 & B\vec{v}_2 & \cdots & B\vec{v}_m \end{pmatrix}$$. To find the product $BA$, we multiply B by each column of A and combine the resulting vectors.
Theorem 2.3.3
Matrix multiplication is non-commutative
$AB \neq BA$, in general. However, at times it does happen that $$AB=BA$$; then we say that $A$ and $B$ commute.
Theorem 2.3.4
The entries of the matrix product
Let $B$ be an $n \times p$ matrix and $A$ be an $p \times m$ matrix. Then, the entry in the $i$-th row and $j$-th column of the product $BA$ is given by: $$(BA)_{ij} = \sum_{k=1}^{p} B_{ik}A_{kj}$$
Theorem 2.3.5
Multiplying with the identity matrix
For an $n \times n$ identity matrix $I_n$, we have: $$AI_n = A$$ $$I_nA = A$$ for any $n \times n$ matrix $A$.
Theorem 2.3.6
Matrix multiplication is associative
Let $A$ be an $n \times p$ matrix and $B$ and $C$ be $p \times m$ matrices. Then: $$A(BC) = (AB)C$$
Theorem 2.3.7
Matrix multiplication is distributive
Let $A$ be an $n \times p$ matrix and $B$ and $C$ be $p \times m$ matrices. Then: $$A(B+C) = AB + AC$$ $$(B+C)A = BA + CA$$
Theorem 2.3.8
Matrix multiplication is compatible with scalar multiplication
Let $A$ be an $n \times p$ matrix and $B$ be a $p \times m$ matrix. Then: $$A(cB) = c(AB) = (cA)B$$ for any scalar $c$.
The Inverse of a linear transformation
Inverse functions
A function $T$ from $X$ to $Y$ is called invertible if the equation $T(\vec{x}) = \vec{y}$ has a unique solution x in X for every y in Y.
In this case, the inverse function $T^{-1}$ from Y to X is defined by $T^{-1}(\vec{y}) = \vec{x}$ the unique x in X such that $T(\vec{x}) = \vec{y}$.
To put it differently, the equation $T(\vec{x}) = \vec{y}$ has a unique solution for every y in Y if and only if the equation $T^{-1}(\vec{y}) = \vec{x}$ has a unique solution for every x in X.
Note that $T^{-1}(T(x))=x$ for all $x$ in $X$ and $T(T^{-1}(y))=y$ for all $y$ in $Y$.
Conversely, if $L$ is a function from $Y$ to $X$ such that $L(T(x))=x$ for all $x$ in $X$ and $T(L(y))=y$ for all $y$ in $Y$, then $L$ is the inverse function of $T$.
If a function $T$ is invertible, then so is $T^{-1}$ and $(T^{-1})^{-1}=T$.
Definition 2.4.2
Invertible matrices
A square matrix $A$ is said to be invertible if the linear transformation $\vec{y}=T(\vec{x})=A\vec{x}$ is invertible. In this case, the matrix of $T^{-1}$ is denoted by $A^{-1}$. If the linear transformation $\vec{y}=T(\vec{x})=A\vec{x}$ is invertible, then its inverse is $\vec{x}=T^{-1}(\vec{y})=A^{-1}\vec{y}$.
Theorem 2.4.3
Invertibility
An $n \times n$ matrix $A$ is invertible if and only if $$rref(A)=I_n$$ or, equivalently, if $$rank(A)=n$$
Theorem 2.4.4
Invertibility and linear systems
Let A be an $n \times n$ matrix: a. Consider a vector $\vec{b}$ in $\mathbb{R}^n$. If A is invertible, then the system $A\vec{x}=\vec{b}$ has a unique solution given by $\vec{x}=A^{-1}\vec{b}$. If A is noninvertible, then the system $A\vec{x}=\vec{b}$ has either no solution or infinitely many solutions.
b. Consider the special case when $\vec{b}=\vec{0}$. The system $A\vec{x}=\vec{0}$ has $\vec{x}=\vec{0}$ as a solution. If A is invertible, then this is the only solution. If A is noninvertible, then the system $A\vec{x}=\vec{0}$ has infinitely many solutions.
Theorem 2.4.5
Finding the inverse of a matrix
To find the inverse of an $n \times n$ matrix A, form the $n \times 2n$ augmented matrix $$[A|I_n]$$ and compute the reduced row echelon form of this matrix.
  • If $rref([A|I_n])$ is of the form $[I_n|B]$, then A is invertible and $A^{-1}=B$.
  • If $rref[A|I_n]$ is of another form , then A is noninvertible. Note that the left half of $rref[A|I_n]$ is $rref(A)$
