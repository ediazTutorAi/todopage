Finding the Eigenvectors of a Matrix

MATH 3330 · Dr. Esteban Diaz
Definition 7.3.1
Eigenspaces
Consider an eigenvalue $\lambda$ of an $n\times n$ matrix A. Then the kernel of the matrix $A-\lambda I_n$ is called the eigenspace associated with $\lambda$, denoted by $E_{\lambda}$: $$E_{\lambda}=ker(A-\lambda I_n)=\{\vec{v}\in\mathbb{R}^n:A\vec{v}=\lambda \vec{v}\}$$
Definition 7.3.2
Geometric multiplicity
Consider an eigenvalue $\lambda$ of an $n\times n$ matrix A. The dimension of eigenspace $E_{\lambda}=ker(A-\lambda I_n)$ is called the geometric multiplicity of eigenvalue $\lambda$, denoted $\text{gemu}(\lambda)$. Thus, $$\text{gemu}(\lambda)=nullity(A-\lambda I_n)=n-rank(A-\lambda I_n)$$
Theorem 7.3.3
Eigenbases and geometric multiplicities
a. Consider an $n\times n$ matrix A. If we find a basis of each eigenspace of A and concatenate all these bases, then the resulting eigenvetors $\vec{v}_1,\cdots,\vec{v}_s$ will be linearly independent. Note that $s$ is the sum of the geometric multiplicities of the eigenvalues of A. This result implies $s\leq n$

b. Matrix A is diagonalizable if and only if the geometric multiplicities of the eigenvalues add up to n. Meaning that $s=n$ in part a.
Theorem 7.3.4
An $n\times n$ matrix with n distinc eigenvalues
If an $n\times n$ matrix A has n distinct eigenvalues, then A is diagonalizable. We can construct an eigenbasis by finding an eigenvector for each eigenvalue.
Theorem 7.3.5
The eigenvalues of similar matrices
Suppose a matrix A is similar to B. Then
a. Matrices A and B have the same characteristic polynomial; that is, $f_A({\lambda})=f_B(\lambda)$.

b. $rank(A)=rank(B)$ and nullity A = nullity B.

c. Matrices A and B have the same eigenvalues, with the same algebraic and geometric multiplicities. However, the eigenvectors need not be the same.

d. Matrices A and B have the same determinant and the same trace:
detA=detB and tr(A)=tr(B).
Theorem 7.3.6
Algebraic versus geometric multiplicity
If $\lambda$ is an eigenvalue of a square matrix A, then $$\text{gemu}(\lambda)=\text{almu}(\lambda)$$
Definition 7.3.7
Strategy for Diagonalization
Suppose we are asked to determine whether a given $n\times n$ matrix A is diagonalizable. If so, we wish to find an invertible matrix S such that $S^{-1}AS=B$ is diagonal.
We can proceed as follows:
a. Find the eigenvalues of A by solving the characteristic equation $$f_A(\lambda)=det(A-\lambda I_n)=0$$ b. For each eigenvalue $\lambda$, find a basis of the eigenspace $$E_{\lambda}=ker(A-\lambda I_n)$$ c. Matrix A is diagonalizable if and only if the dimensions of the eigenspace add up to n. In this case, we find an eigenbasis $\vec{v}_1,\cdots,\vec{v}_n$ for A by concatenating the bases of the eigenspaces we found in part b. Let $$S=\begin{pmatrix}|&|&\cdots&|\\\vec{v}_1&\vec{v}_2&\cdots&\vec{v}_n\\|&|&\cdots&|\end{pmatrix}$$ Then $S^{-1}AS=B=diag(\lambda_1,\lambda_2,\cdots,\lambda_n$
Theorem 7.2.7
Number of eigenvalues
An $n\times n$ matrix has at most n real eigenvalues, even if they are counted with their algebraic multiplicities.
If n is odd, then an $n\times n$ matrix has at least one real eigenvalue
Theorem 7.2.8
Eigenvalues, determinant, and trace
If an $n\times n$ matrix A has the eigenvalues $\lambda_1,\lambda_2,\cdots,\lambda_n$ listed with their algebraic multiplicities, then: $$det A = \lambda_1\lambda_2\cdots\lambda_n$$ the product of the eigenvalues, and
$$tr A=\lambda_1+\lambda_2+\cdots\lambda_n$$, the sum of the eigenvalues.
