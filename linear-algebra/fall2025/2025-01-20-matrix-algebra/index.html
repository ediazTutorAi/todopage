On the Solutions of Linear Systems

MATH 3330 · Dr. Esteban Diaz
Theorem 1.3.1
Number of solutions of a linear system
A system of equations is said to be consistent if there is at least one solution. Otherwise, it is said to be inconsistent.
A linear system is inconsistent if and only if the reduced row-echelon form has a row of the form $\begin{bmatrix}0&0&\cdots&0&1\end{bmatrix}$, representing the equation $0=1$.
If a linear system is consistent, then it has either:
  • Unique solution: the reduced row-echelon form has a leading 1 in every column of the coefficient matrix.
  • Infinitely many solutions: the reduced row-echelon form has a leading 1 in every column of the coefficient matrix, but some columns do not have a leading 1, in other words, it has free variables.
Definition 1.3.2
The rank of a matrix
The rank of a matrix A is the number of leading 1's in rref(A), denoted as rank(A).
Example 3
Consider a system of n linear equations with m variables, which has a coefficient matrix A of size $n\times m$. Show that:
  • a. The inequalities $rank(A) \leq n$ and $rank(A) \leq m$ always hold.
  • b. If the system is inconsistent, then $rank(A) < n $.
  • c. If the system is consistent, then $rank(A) = n$.
  • d. If the system has exactly one solution, then $rank(A) = m$.
  • e. If the system has infinitely many solutions, then $rank(A) < m$.
Example 4
It is useful to think about the contrapositive of the previous statements:
  • b. $rank(A) = n$, then the system is consistent.
  • c. $rank(A) = m$, then the system has no solution or exactly one solution.
  • d. $rank(A) < m$ then the system has no solution or infinitely many solutions.
Theorem 1.3.3
Number of equations vs. number of unknowns
a. If a linear system has one solution, then there must b at least as many equations as there are variables.
Equivalently, we can formulate the contrapositive:
b. A linear system with fewer equations than unknowns has either no solutions or infinitely many solutions.
Theorem 1.3.4
System of n equations in n variables
A linear system of n equations in n variables has a unique solution if and only if the rank of its coefficient matrix A is n. In this case: $$rref(A)=1_{n\times n}$$ the nxn matrix with 1's along the diagonal and 0's everywhere else.
Matrix Algebra
Definition 1.3.5 Sums of matrices: If $A$ and $B$ are $m\times n$ matrices, then their sum $A+B$ is the $m\times n$ matrix obtained by adding corresponding entries: $$(A+B)_{ij}=A_{ij}+B_{ij}$$ Scalar multiple of matrices: If $A$ is an $m\times n$ matrix and $c$ is a scalar, then the scalar multiple $cA$ is the $m\times n$ matrix obtained by multiplying each entry of $A$ by $c$: $$(cA)_{ij}=cA_{ij}$$
Definiton 1.3.6
Dot product of vectors
Consider two vectors $\vec{u}$ and $\vec{w}$ with components $(u_1, u_2, \ldots, u_n)$ and $(w_1, w_2, \ldots, w_n)$, respectively. The dot product of $\vec{u}$ and $\vec{w}$ is defined as: $$\vec{u} \cdot \vec{w} = u_1 w_1 + u_2 w_2 + \cdots + u_n w_n$$ Note that our definition of the dot product is not row-column sensitive. The dot product does not distinguish between row and column vectors.
Definition 1.3.7
The product of $A\vec{x}$
If A is an nxn matrix with row vectors $\vec{a}_1, \vec{a}_2, \ldots, \vec{a}_n$ and $\vec{x}$ is an n-dimensional column vector with components $x_1, x_2, \ldots, x_n$, then the product $A\vec{x}$ is defined as: $$A\vec{x} = \begin{pmatrix} \vec{a}_1 \\ \vec{a}_2 \\ \vdots \\ \vec{a}_n \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{pmatrix}$$ In words, the ith component of $A\vec{x}$ is the dot product of the $i$th row of $A$ with $\vec{x}$. Note that $A\vec{x}$ is a column vector with $n$ components, that is, a vector in $\mathbb{R}^n$.
Theorem 1.3.8
The product $A\vec{x}$ in terms of the columns of $A$
If the column vectors of an nxn matrix A are $\vec{v_1}, \vec{v_2}, \ldots, \vec{v_n}$, then the product $A\vec{x}$ can be expressed as: $$A\vec{x} = x_1\vec{v_1} + x_2\vec{v_2} + \cdots + x_n\vec{v_n}$$
Definition 1.3.9
Linear combinations
A vector $\vec{b}$ in $\mathbb{R}^n$ is a linear combination of the vectors $\vec{v_1}, \vec{v_2}, \ldots, \vec{v_n}$ in $\mathbb{R}^n$ if there exist scalars $x_1, x_2, \ldots, x_n$ such that: $$\vec{b} = x_1\vec{v_1} + x_2\vec{v_2} + \cdots + x_n\vec{v_n}$$ Give examples.
Theorem 1.3.10
Algebraic rules for $A\vec{x}$
If A is an nxn matrix, $\vec{x}$ and $\vec{y}$ are vectors in $\mathbb{R}^n$, and $k$ is a scalar, then: $$A(\vec{x} + \vec{y}) = A\vec{x} + A\vec{y}$$ $$A(k\vec{x}) = k(A\vec{x})$$
Theorem 1.3.11
Matrix form of a linear system
We can write the linear system with augmented matrix $[A|\vec{b}]$ in matrix form as $$A\vec{x}=\vec{b}$$.
