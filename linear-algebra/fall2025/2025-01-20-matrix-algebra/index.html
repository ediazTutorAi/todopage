On the Solutions of Linear Systems

MATH 3330 · Dr. Esteban Diaz
Theorem 1.3.1
Number of solutions of a linear system
A system of equations is said to be consistent if there is at least one solution. Otherwise, it is said to be inconsistent.
A linear system is inconsistent if and only if the reduced row-echelon form has a row of the form $\begin{bmatrix}0&0&\cdots&0&1\end{bmatrix}$, representing the equation $0=1$.
If a linear system is consistent, then it has either:
  • Unique solution: the reduced row-echelon form has a leading 1 in every column of the coefficient matrix.
  • Infinitely many solutions: the reduced row-echelon form has a leading 1 in every column of the coefficient matrix, but some columns do not have a leading 1, in other words, it has free variables.
Definition 1.3.2
The rank of a matrix
The rank of a matrix A is the number of leading 1's in rref(A), denoted as rank(A).
Example 3
Consider a system of n linear equations with m variables, which has a coefficient matrix A of size $n\times m$. Show that:
  • a. The inequalities $rank(A) \leq n$ and $rank(A) \leq m$ always hold.
  • b. If the system is inconsistent, then $rank(A) < n $.
  • c. If the system is consistent, then $rank(A) = n$.
  • d. If the system has exactly one solution, then $rank(A) = m$.
  • e. If the system has infinitely many solutions, then $rank(A) < m$.
Example 4
It is useful to think about the contrapositive of the previous statements:
  • b. $rank(A) = n$, then the system is consistent.
  • c. $rank(A) = m$, then the system has no solution or exactly one solution.
  • d. $rank(A) < m$ then the system has no solution or infinitely many solutions.
Theorem 1.3.3
Number of equations vs. number of unknowns
a. If a linear system has one solution, then there must b at least as many equations as there are variables.
Equivalently, we can formulate the contrapositive:
b. A linear system with fewer equations than unknowns has either no solutions or infinitely many solutions.
Consistent vs. inconsistent, page 6
A system is consistent if it has at least one solution. It is inconsistent if the equations contradict each other. A classic contradiction pattern is something like $$0=1,$$ which signals “no solution.”
Coefficient viewpoint, page 7
The numbers $a_{ij}$ are the coefficients, the $x_j$ are the unknowns, and the $b_i$ form the right-hand side. It’s helpful to separate:
  • Structure: the coefficient pattern $(a_{ij})$
  • Targets: the numbers $(b_i)$
Preview: matrix form, page 8
A linear system can be packaged as $$A\mathbf{x}=\mathbf{b},$$ where $A$ is the coefficient matrix, $\mathbf{x}$ is the vector of unknowns, and $\mathbf{b}$ is the right-hand side. (The mechanics of solving this efficiently comes next.)
Why linear systems matter, page 9
Linear systems show up everywhere: balancing flows, fitting data, networks, economics, circuits, and as approximations to nonlinear problems. Many later topics (linear transformations, subspaces, eigenvalues) build on understanding $$A\mathbf{x}=\mathbf{b}.$$
Common student traps, page 10
  • Solving equations one-by-one (instead of simultaneously).
  • Assuming “two equations means one unique solution.” (Not true.)
  • Missing that dependent equations can produce infinitely many solutions.
  • Thinking inconsistency is rare—contradictions happen often in models/data.
Problem 1, page 11
Consider the following systems of linear equations:

(a) $$\begin{cases} x + 2y = 3\\ 2x + 4y = 6 \end{cases}$$
(b) $$\begin{cases} x + 2y = 3\\ 2x + 4y = 5 \end{cases}$$
(c) $$\begin{cases} x + 2y = 3\\ 2x - y = 1 \end{cases}$$

For each system, determine whether it has no solution, exactly one solution, or infinitely many solutions.
Problem 2, page 12
Geometric interpretation
Each equation below represents a plane in $\mathbb{R}^3$: $$\begin{cases} x + y + z = 1\\ 2x + 2y + 2z = 2\\ x - y + z = 0 \end{cases}$$
Without solving the system explicitly, describe the geometric relationship among the planes and predict the number of solutions to the system.
Problem 3, page 13
Modeling with linear systems
A student buys tickets for a concert. Adult tickets cost \$12 and student tickets cost \$8. The student buys a total of 15 tickets and spends \$152.

(a) Define appropriate variables. (b) Write a system of linear equations that models the situation. (c) Explain what it would mean if the system were inconsistent.
Example
Consider the system of linear equations: $$\begin{cases} x + y + z = 3 \\ 2x + 2y + 2z = 7 \\ x + y + z = 4 \end{cases}$$
Use equation operations (adding multiples of equations or subtracting equations) to eliminate variables and determine whether the system has a solution.

Explain what the final contradiction means in terms of the original equations.
