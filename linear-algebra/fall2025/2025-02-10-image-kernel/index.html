3.1 Image and Kernel of Linear Transformations

Image and Kernel of a Linear Transformation
Definition 3.1.1
The image of a function consists of all the values the function takes in its target space. If f is a function from X to Y, then: $$image(f)=\{f(x):x in X\}=\{b in Y:b=f(x) for some x in X\}$$
Definition 3.1.2
Span
Consider the vectors $\vec{v}_1, \vec{v}_2, ..., \vec{v}_n$ in $\mathbb{R}^n$. The set of all linear combinations $c_1\vec{v}_1 + c_2\vec{v}_2 + ... + c_n\vec{v}_n$ of vectors $\vec{v}_1, \vec{v}_2, ..., \vec{v}_n$ is called their span: $$span(\vec{v}_1, \vec{v}_2, ..., \vec{v}_n) = \{c_1\vec{v}_1 + c_2\vec{v}_2 + ... + c_n\vec{v}_n: c_1, c_2, ..., c_n in \mathbb{R}\}$$
Theorem 3.1.3
Image of a Linear Transformation
Let T: $\mathbb{R}^n \to \mathbb{R}^m$ be a linear transformation. The image of T is the span of the columns of the matrix A associated with T.
Theorem 3.1.4
Some properties of the image of a linear transformation
The image of a linear transformation T: $\mathbb{R}^n \to \mathbb{R}^m$ has the following properties:
  1. The zero vector $\vec{0}$ is in the image of T.
  2. If $\vec{u}$ and $\vec{v}$ are in the image of T, then $\vec{u} + \vec{v}$ is in the image of T.
  3. If $\vec{u}$ is in the image of T and c is a scalar, then $c\vec{u}$ is in the image of T.

The Kernel of a linear transformatiion
Definiton 3.1.5
The kernel of a linear transformation $T(\vec{x})=A\vec{x}$ from $\mathbb{R}^m$ to $\mathbb{R}^n$ consists of all zeros of the transformation, that is, the solutions of the equation $T(\vec{x})=\vec{0}$.
We denote the kernel of T by $ker(T)$ or $ker(A)$

For a linear transformation T from $\mathbb{R}^m$ to $\mathbb{R}^n$.
  1. $im(T)=\{T(\vec{x}):\vec{x} in \mathbb{R}^m\}$ is a subset of the target space $\mathbb{R}^n$ of T
  2. $ker(T)=\{x in \mathbb{R}^m:T(\vec{x})=\vec{0}\}$ is a subset of the domain $\mathbb{R}^m$ of T
Theorem 3.1.6
Some properties of the kernel
Considefr a linear transformation $T: \mathbb{R}^m \to \mathbb{R}^n$. The kernel of T has the following properties:
  1. The zero vector $\vec{0}$ is in the kernel of T.
  2. The kernel is closed under addition
  3. The kernel is closed under scalar multiplication
Theorem 3.1.7
When is $ker(A)={\vec{0}}$
  1. Consider an $n \times n$ matrix A. Then $ker(A)={\vec{0}}$ if and only if $rank(A)=m$
  2. Consider an $n \times m$ matrix A. If $ker(A)={\vec{0}}$, then $m \leq n$. Equivalently, if $m > n$, then there are nonzero vectors in $ker(A)$
  3. For a square matrix A, we have $ker(A)={\vec{0}}$ if and only if $A$ is invertible
Theorem 3.1.8
Various characterizations of invertible matrix
For an $n \times n$ matrix A, the following statements are equivalent; that is, for a given A, they are either all true or all false:
  1. A is invertible
  2. The linear system $A\vec{x}=\vec{b}$ has a unique solution for every $\vec{b}$ in $\mathbb{R}^n$
  3. $rref(A)=I_n$
  4. The rank of A is n
  5. $im(A)=\mathbb{R}^n$
  6. $ker(A)=\{\vec{0}\}$
