5.6 The root Test

Theorem 5.17
Root Test
Consider the series \(\displaystyle\sum_{n=1}^{\infty}a_n\). Let \[\rho=\lim_{n\to\infty}\sqrt[n]{|a_n|}\]
  1. If \(0\leq\rho < 1\), then \(\displaystyle\sum_{n=1}^{\infty}a_n\) converges
  2. If \(\rho > 1\) or \(\rho=\infty\), then \(\displaystyle\sum_{n=1}^{\infty}a_n\) diverges
  3. If \(\rho=1\), the test does not provide any information
