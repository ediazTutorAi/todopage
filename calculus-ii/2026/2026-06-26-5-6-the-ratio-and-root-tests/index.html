Theorem 5.16
Ratio Test
Let \(\displaystyle\sum_{n=1}^{\infty}a_n\) be a series with nozero terms. Let \[\rho=\lim_{n\to\infty}\left|\frac{a_{n+1}}{a_n}\right|\]
  1. If \(0\leq\rho < 1\) , then \(\displaystyle\sum_{n=1}^{\infty}a_n\) converges absolutely
  2. If \(\rho > 1\) or \(\rho = \infty\), then \(\displaystyle\sum_{n=1}^{\infty}a_n\) diverges
  3. If \(\rho=1\), the test does not provide any information
Using the Ratio Test
For each of the following series, use the ratio test to determine whether the series converges or diverges.
  1. \(\displaystyle\sum_{n=1}^{\infty}\frac{2^n}{n!}\)
  2. \(\displaystyle\sum_{n=1}^{\infty}\frac{n^n}{n!}\)
  3. \(\displaystyle\sum_{n=1}^{\infty}\frac{(-1)^n(n!)^2}{(2n)!}\)
Warning
Pedagogical Issues when using the ratio Test
  1. Choosing the Ratio Test when another test is obvious. Think of \(\displaystyle\sum{n=1}^{\infty}\frac{1}{n^2}\)
  2. Thinking that the limit of the ratio equals 1 means divergence
  3. Forgetting the absolute value
  4. Simplification mistakes
  5. Mishandling powers
  6. Difficult with factorials
  7. Limits involving infinity, using the LHospital Rule
